pi-web

Install pi-web in one command.

Minimal install for a browser-based web agent.

GitHub npm website source
Main command
npm install -g @dimko33/pi-web && pi-web
Direct install (no npm)
curl -fsSL https://raw.githubusercontent.com/dimko33-lang/pi-web-agent/main/bootstrap.sh | bash

After install you will see

Admin panel: ?admin
Client panels: ?default, ?slot1, ?slot2
