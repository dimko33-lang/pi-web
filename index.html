pi-web

Install pi-web in one command.

Minimal install options for a browser-based web agent.

GitHub npm website source
Recommended

Shell install

Best option for a clean Linux server. No Node.js required.

bash <(curl -fsSL https://raw.githubusercontent.com/dimko33-lang/pi-web-agent/main/bootstrap.sh)
Minimal

npm install

Use this if Node.js and npm are already installed.

npm install -g @dimko33/pi-web
Ubuntu / Debian

One-liner

Installs Node.js first, then installs pi-web globally.

curl -fsSL https://deb.nodesource.com/setup_22.x | bash - && apt-get install -y nodejs && npm install -g @dimko33/pi-web
