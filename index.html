pi-web

Install pi-web in one command.

Minimal install options for a browser-based web agent.

npm package: @dimko33/pi-web · current published version: 0.1.3

GitHub npm website source
Main command
npm install -g @dimko33/pi-web
During setup, the installer asks for Public host/IP and then key prompts for OpenRouter, GROQ, Gemini, Anthropic, OpenAI, and Kimi. If you do not have a key for a provider, just press Enter.
If Node.js is not installed

Ubuntu / Debian one-liner

Installs Node.js first, then installs pi-web globally.

curl -fsSL https://deb.nodesource.com/setup_22.x | bash - && apt-get install -y nodejs && npm install -g @dimko33/pi-web
Direct shell install

No npm required

Use this if you want to install directly from the bootstrap script.

bash <(curl -fsSL https://raw.githubusercontent.com/dimko33-lang/pi-web-agent/main/bootstrap.sh)

Setup notes

Current built-in providers in the app: OpenRouter, GROQ, and Kimi.

Extra keys can also be stored during setup now: Gemini, Anthropic, and OpenAI.

  • If you do not have a key for a provider, just press Enter.
  • To complete install, you need at least one currently supported key: OpenRouter, GROQ, or Kimi.
  • At the end of install, the terminal prints the web links you need.
  • After setup, most interaction happens through the web interface, not the terminal.
Early version. Minimal, practical, and built for real use first.
