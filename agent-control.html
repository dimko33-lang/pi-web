Useful commands

Run these on your server (as root or sudo).

Reset page to default (fix broken UI)
sqlite3 /opt/my-agent/agent.db "DELETE FROM session_html;" && systemctl restart my-agent
Resets HTML to default, restores all buttons and interface.
Restart agent service
systemctl restart my-agent
systemctl status my-agent --no-pager
Increase answer length (max_tokens = 16384)
sed -i 's/"max_tokens": [0-9]*/"max_tokens": 16384/' /opt/my-agent/agent.py && systemctl restart my-agent
Adjust creativity (temperature)
sed -i 's/"temperature": [0-9.]*/"temperature": 1.5/' /opt/my-agent/agent.py && systemctl restart my-agent
sed -i 's/"temperature": [0-9.]*/"temperature": 0.5/' /opt/my-agent/agent.py && systemctl restart my-agent
Higher = more creative/chaotic, lower = more predictable/focused.
Change AI model (Groq)
echo '{"provider": "groq", "model": "llama-3.3-70b-versatile"}' > /opt/my-agent/global_model.json && systemctl restart my-agent
Available models: llama-3.1-8b-instant, llama-3.3-70b-versatile, mixtral-8x7b-32768, kimi-k2, etc.
Update API keys (GROQ, OpenRouter, Kimi)
nano /etc/my-agent.env
systemctl restart my-agent
After editing keys, restart the service.
Full reinstall (keeps server)
systemctl stop my-agent; rm -rf /opt/my-agent /root/pi-web-agent-src; cd /root; git clone https://github.com/dimko33-lang/pi-web-agent.git /root/pi-web-agent-src; cd /root/pi-web-agent-src; bash install.sh
View logs (errors)
journalctl -u my-agent -n 50 --no-pager
Backup current HTML (before changes)
sqlite3 /opt/my-agent/agent.db "SELECT html FROM session_html WHERE session_id=1;" > /root/page_backup.html
Full root access (shell commands)

Use !shell <command> in the agent chat to execute any system command as root. Examples:

!shell apt update && apt upgrade -y
!shell systemctl restart my-agent
!shell ls -la /opt/my-agent
!shell cat /etc/my-agent.env
⚠️ These commands run as root – be careful!
